##1 变量

###1 变量的初始化
    C++的变量初始化有两种方式：复制初始和直接初始化。复制初始化语法用等号（=）
    直接初始化则是把初始化表达式放在括号中

    ```C++
        int ival(1024);  //direct init
        int ival = 1024; // copy init 
    ```

    **初始化不是赋值**
